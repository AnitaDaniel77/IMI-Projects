Semantic HTML Cheat Sheet

Sectioning Tags

<header>
Logo and branding at the top of the page
<nav>
Navigation links
<main>
Primary content of the page
<aside>
Secondary content like a sidebar
<section>
Standalone section within the page
<article>
Self-contained content like a blog post
<footer>
Copyright and secondary links
<details> and <summary>
Collapsible content
<h1> to <h6>
Headings from most to least important

Content Tags

<p>
Paragraph of text
<ul>
Unordered list
<ol>
Ordered list
<li>
List item
<blockquote>
Quotation
<figure> and <figcaption>
Image with caption
<hr>
Horizontal dividing line
<pre>
Preformatted monospace text
<dl>, <dt>, <dd>
Description list with terms and definitions

Inline Tags

<a>
Anchor link to another page
<strong>
Bold text indicating importance
<em>
Emphasised italic text
<span>
Generic grouping for CSS styling
<mark>
Highlighted text
<time>
Semantic dates and times
<code>
Inline computer code
<small>
Copyright and legal text
<br>
Line break
<abbr>
Abbreviation or acronym
<sub> and <sup>
Subscript and superscript text

Media Tags

<img>
Embeds an image
<video>
Embeds a video player
<audio>
Embeds an audio player
<iframe>
Embeds another webpage
<picture>
Responsive images for different devices
<svg>
Scalable vector graphics
<canvas>
2D and 3D graphics
<source>
Media source for video, audio and picture

Table Tags

<table>
Table container
<thead>
Table header section
<tbody>
Table body section
<tfoot>
Table footer section
<tr>
Table row
<th>
Header cell
<td>
Data cell
<caption>
Table caption
