Lesson 22 — Figma: Frames, Layers and Shapes

Three core concepts

A frame is a container where your design lives — equivalent to an Artboard in other tools. Think of it as a screen in an app. Screen 1, Screen 2, and so on. Each frame is independent and can be sized separately.

A layer is any object that lives inside a frame. Every time you add an object, Figma creates a new layer for it automatically.

A group combines multiple objects into one. Selecting them and hitting Ctrl/Cmd G wraps them in a single parent layer. The difference between a group and a frame is that frames have more functionality — independent sizing being the main one you'll use later.

Creating a frame

Click the frame icon in the top toolbar, then click and drag on the canvas. Alternatively, select the frame icon and choose a preset from the right panel — desktop, mobile, tablet and so on.

To zoom: Ctrl/Cmd + to zoom in, Ctrl/Cmd - to zoom out. To pan: select the hand icon in the toolbar or hold Shift.

Drawing shapes

Select the frame first, then open Shape Tools from the toolbar by clicking the arrow next to the circle icon. To draw a square, select the rectangle tool and hold Shift while dragging. Same for a perfect circle — select ellipse and hold Shift.

To duplicate any shape: select it and press Ctrl/Cmd D. The copy lands on top and can be dragged wherever you need it.

Shape properties

Select a shape and the right panel shows its x and y coordinates, width, height, rotation and corner radius. Changing corner radius to 100 on a square turns it into a pill or circle depending on its dimensions.

Every shape defaults to a grey fill with no stroke. To change the fill, click the colour swatch in the Fill section — that opens the colour palette where you can pick a colour, drag the hue slider, adjust transparency or type in a hex value directly.

Stroke is the outline. You can set it to inside, outside or center, and change its line type in the advanced stroke settings. Effects like drop shadow are added via the Effects section — click the plus icon to add one.

Working with layers

The layers panel on the left lists every object in the order it was drawn. The top of the list sits visually on top of the canvas. To reorder, drag a layer above or below another in the panel.

Name your layers properly. Renaming "Rectangle 2" to "square" and "Ellipse 1" to "circle" keeps the file readable as it grows — especially when you're working with dozens of elements.

You can toggle a layer's visibility on and off using the eye icon, or lock it so it can't be accidentally moved.

Grouping

Select multiple objects, right-click and choose Group Selection, or press Ctrl/Cmd G. The group moves and scales as one object. To edit an individual shape inside the group, double-click into it.
