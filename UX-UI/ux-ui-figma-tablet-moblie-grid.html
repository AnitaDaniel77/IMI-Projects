Lesson 27 — Figma: Tablet and Mobile Grids

Once your desktop grid is set up, creating tablet and phone grids follows the same process — you're mostly copying and adjusting column counts.

Tablet grid

Press F to open the frames panel and select an iPad Mini from the tablet menu. To copy the grid from your desktop frame, hover just left of the grid icons in the right sidebar until a stack icon appears. Hold Ctrl or Shift, click it to select the grid, then copy and paste it into the tablet frame. Change the 12-column grid down to 8 columns. That gives content enough room to breathe on a smaller canvas.

Phone grid

Same process. Select a phone frame, paste the grid in, change to 4 columns and set the margin to 20. Four columns is the standard for mobile — it keeps content in a single, readable flow without cramming elements side by side.

You can stretch any of these frames and they'll scale accordingly. The grids are a starting point, not a fixed rule, and get more detailed as your designs become more specific to particular devices.

Hiding the grid

To see the frame without the grid: click the zoom percentage top-right to open View Settings and toggle grids off. Keyboard shortcut is Shift G on Mac, Ctrl Shift 4 on Windows. Same shortcut brings it back.

Nudge settings

A nudge is how far a layer moves when you press an arrow key. Small nudge moves 1px at a time. Big nudge uses Shift + arrow key and defaults to 10px in Figma, which doesn't align with the 8-point grid. Change it to 8: go to the Figma icon top-left, select Preferences, then Nudge Amount, and set the large nudge to 8. Now every keyboard move stays on-grid without manual checking.
