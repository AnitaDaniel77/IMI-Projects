Lesson 24 — Figma: Text and Fonts

Two ways to create a text element: click the T icon in the toolbar (or press T), then click once to start typing, or click and drag to set a fixed box size. The difference matters — a single click creates an auto-resize box that grows horizontally as you type, while click-and-drag locks the width so text wraps to the next line at the edge.

To edit existing text, double-click the box. To change resize behaviour, go to the design panel on the right and adjust the auto-resize setting. The three modes are grow horizontally (default on click), fixed (default on click-and-drag), and grow vertically, where the box expands downward as content increases but stays the same width.

Fonts and styling

The font dropdown lives in the design panel. You can browse or type to search — searching "Oswald" for example filters straight to it. Style variants (bold, italic, light) sit in a second dropdown next to the font name. Keyboard shortcuts work too: Ctrl/Cmd B for bold, Ctrl/Cmd I for italic, Ctrl/Cmd U for underline.

Font size can be typed in directly or nudged with the arrow keys. Line height defaults to auto — increasing it (say, to 43) adds breathing room between lines. Character spacing (kerning) sits next to line height and accepts either percentage or pixel values.

Alignment

Horizontal alignment controls how text sits within its bounding box: left, center, right. Vertical alignment controls top, middle, or bottom distribution inside the box. Both live in the text properties panel.

Advanced text options

Clicking the three dots at the bottom of the text section opens three tabs. Basics covers resize behaviour, decoration (strikethrough, underline), paragraph spacing, indentation, list formatting, and text case. Details adds superscript, subscript, fractions, and OpenType features like alternate letterforms and character variants. Variables handles variable font axis settings if the font supports them.

Other text attributes

Text layers accept the same treatments as shapes. You can change the fill colour, add a stroke, or apply effects like drop shadow — all from the same right panel sections used for any other object.
