Lesson 37 — Wireframe to High-Fidelity Mockup

A high-fidelity mockup closely resembles the final product. You take your approved wireframe and apply branding — colours, typography, images — using the brand style guide and UI kit provided.

What a brand style guide includes

  • Typefaces and text styles
  • Colour palette
  • Rules for how images should be used
  • A UI kit — font sizes, icons, components and documentation

Creating text styles from the UI kit

Open the UI kit in Figma. Select the display title text. Click the four dots icon on the right sidebar, select Style, click the plus icon, name it "Display" and click Create Style. Repeat this for every text style in the kit.

Creating colour styles

Select a colour swatch in the UI kit. Go to the Fill section, click the four dots, click the plus icon, name it (e.g. "Primary 1" for green, "Primary 2" for yellow) and click Create Style. Now these colours are saved and reusable across the whole file.

Applying styles to the wireframe

Select the background rectangle on the first wireframe frame. In the Fill section click the four dots and choose Primary 1 (green). The background updates instantly.

Select the "Little Lemon" heading text. In the right sidebar under text properties, click the four dots and select the Display style. Change the colour to yellow. Change other text elements to white.

Select the button. In the right sidebar change the corner radius to 16 for rounded edges as specified in the UI kit. Change the fill to Primary 2. Change the button text colour to black — white text on a light button isn't legible.

Adding images

Select an image placeholder rectangle. In the Fill section, change the dropdown from Solid to Image. A checkered box appears prompting you to choose a file. Select your image and it fills the placeholder. Repeat for every image placeholder in the wireframe.

Key takeaway

The process is: wireframe approved → open UI kit → create text and colour styles → apply styles to wireframe elements → swap placeholder rectangles for real images. The structure stays the same, the branding brings it to life.
