React.cloneElement and React.Children.map

Real life example: Like a photocopier that can copy a document and add a stamp to the copy — original stays unchanged, copy gets the new addition.

React.cloneElement — clone and modify an element

```jsx // Clone element and add new props React.cloneElement(element, { newProp: value }) // Example — add style to a child element React.cloneElement(child, { style: { marginLeft: '32px' } }) ```

React.Children.map — iterate over children

```jsx // Similar to array .map() but for children prop React.Children.map(children, child => { // transform each child return React.cloneElement(child, { newProp: value }) }) ```
