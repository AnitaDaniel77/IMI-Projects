useReducer Hook

Real life example: Like a bank account with multiple transaction types — deposit, withdraw, bonus. Instead of managing each separately with useState, useReducer handles all actions in one place.

useState vs useReducer

```jsx // useState — simple, one value const [money, setMoney] = useState(100) // useReducer — better for complex state with multiple actions const [state, dispatch] = useReducer(reducer, initialState) ```

useReducer structure

Custom Hooks

Real life example: Like a reusable recipe. Instead of figuring out how to make the same sauce from scratch every time, you write it down once and use it in any dish.

Why custom hooks?

Extract reusable logic — avoid duplication across components and projects.

Rules for custom hooks

Name must start with "use" — e.g. useConsoleLog, useFetch, useTheme

Custom hook — useConsoleLog.js

```jsx // useConsoleLog.js — logs any value when it changes import { useEffect } from 'react' function useConsoleLog(varName) { useEffect(() => { console.log(varName) }, [varName]) } export default useConsoleLog ```

Using the custom hook in App.js

```jsx import { useState } from 'react' import useConsoleLog from './useConsoleLog' function App() { const [count, setCount] = useState(0) // use custom hook — logs count whenever it changes useConsoleLog(count) function increment() { setCount(count + 1) } return (

Count: {count}

) } export default App ```
