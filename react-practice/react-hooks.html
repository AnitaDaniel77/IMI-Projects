React Hooks

What are Hooks?

Functions introduced in React 16.8 that let you hook into React state and lifecycle features from components.

Real life example: Like a TV remote control — hooks give you built-in buttons to control your component without building the electronics yourself.

Why use Hooks?

They solve the problem of unnecessary code reuse across components and make code more readable and simple.

useState — most common hook

```jsx // Step 1 — import useState from React import { useState } from 'react' // Step 2 — declare state variable using array destructuring const [showMenu, setShowMenu] = useState(false) // showMenu = current state value (starts as false) // setShowMenu = function to update the state // false = initial value ```

Without array destructuring — why we don't do this

```jsx // Long and confusing — avoid this const showMenuArray = useState(false) const showMenu = showMenuArray[0] const setShowMenu = showMenuArray[1] // Clean and simple — use this instead const [showMenu, setShowMenu] = useState(false) ```

useState can track any data type

```jsx const [name, setName] = useState("Anita") // string const [count, setCount] = useState(0) // number const [isOpen, setIsOpen] = useState(false) // boolean const [items, setItems] = useState([]) // array const [user, setUser] = useState({}) // object ```

Tracking button clicks example

```jsx import { useState } from 'react' function App() { const [count, setCount] = useState(0) function handleClick() { setCount(count + 1) } return (

Button clicked {count} times

) } ```

Light switch example

```jsx import { useState } from 'react' function LightSwitch() { const [lightOn, setLightOn] = useState(false) function handleClick() { setLightOn(!lightOn) // ! flips true to false and false to true } return (

{lightOn ? "Light is ON" : "Light is OFF"}

) } ```

Key rules

Call useState at the top level of your component — never inside loops or conditions.

You can also build your own custom hooks to reuse logic across components.

useState with text input

```jsx import { useState } from 'react' export default function InputComponent() { const [inputText, setInputText] = useState('hello') function handleChange(event) { // event.target.value = whatever user typed setInputText(event.target.value) } return (

You typed: {inputText}

) } ```

) } ```

Hook rules

1. Only call hooks at the top level of your component

2. Never call hooks inside loops or conditions

3. Only call hooks from React functions — not regular JavaScript functions
