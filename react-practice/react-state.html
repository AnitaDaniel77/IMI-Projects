React State

What is State?

State is a component's internal data that determines its current behaviour.

Real life example: Like an alarm clock — on, off, snooze are built-in modes. Push a button to change the state.

Props vs State

Props = data from outside, parent controls it, read only

State = data from inside, component controls it, can change

Props vs State — Key Differences

Real life example: Like restaurant blueprints. The blueprint (component) is the same for all locations. Props are the specific configurations each manager chooses (tables, chairs). State is what changes during the day (number of customers seated right now).
