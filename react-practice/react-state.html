React State

What is State?

State is a component's internal data that determines its current behaviour. Like an alarm clock with built-in modes — on, off, snooze — no extra hardware needed, just push a button.

Props vs State recap


Props  = data from outside  — parent controls it
State  = data from inside   — component controls it

Stateless component


// No state — just renders static content
function App() {
    return (
        <h1>A stateless component</h1>
    )
}

Stateful component — useState hook


import React, { useState } from 'react'

function App() {
    // useState returns array of two things:
    // 1. the state value
    // 2. function to update the state
    const [greet, setGreet] = useState("Hello")

    return (
        <h1>{greet}</h1>
    )
}

Array destructuring — how useState works


// Regular array destructuring in JavaScript
const fruits = ["apple", "pear", "plum"]
const [fruit1, fruit2, fruit3] = fruits
// fruit1 = "apple", fruit2 = "pear", fruit3 = "plum"

// useState works the same way
const [greet, setGreet] = useState("Hello")
// greet    = "Hello"   (the state value)
// setGreet = function  (used to update greet)

Naming convention


// State variable name    → update function name
const [greet, setGreet]   = useState("Hello")
const [darkMode, setDarkMode] = useState(false)
const [count, setCount]   = useState(0)
// Pattern: [name, setName]

Key points


1. State is internal to the component
2. When state changes React re-renders the component
3. State is initialised when component is created
4. useState is a React Hook
5. Parent sends state to children via props
6. Components can be stateful or stateless

DJ Lopes website example


import { useState } from 'react'

function BookingButton() {
    // State — tracks if booking form is open
    const [isOpen, setIsOpen] = useState(false)

    return (
        <div>
            <button onClick={() => setIsOpen(!isOpen)}>
                {isOpen ? "Close Form" : "Book DJ Lopes"}
            </button>
            {isOpen && <p>Booking form here</p>}
        </div>
    )
}
